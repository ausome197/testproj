## Instructions for running it locally:

* Install the packages using
`npm install`

* Run the server:
`npm run server`
***
<span style="color:#778800;font-size:1.6em;font-weight:Bold">Description:</span>
<br>
* Frame work used: Express JS.
* Basic Get query API.
* Routing implemented.
* `/user/:username` will display entered <span style="color:#333388">*username*</span> after '/' .
* `/date` will display current date.
